# Workshop Presentations

## Main Workshop Materials (2 Sessions)

### Day 1: Industry Standards & Foundational Patterns
**File:** [SESSION-1-industry-standards.md](SESSION-1-industry-standards.md)

**Duration:** 90 minutes
**Focus:** ADRs + Foundational prompt patterns (Persona, Few-shot, Template, Chain-of-Thought)

**Demo repo branch:** [demo-day-1](https://github.com/josephrobertlopez/spring-migration-demo/tree/demo-day-1)

**Learning Path:**
1. Industry-standard decision documentation (ADRs)
2. Configuration management (`.github/copilot-instructions.md`)
3. Foundational prompt patterns
4. 5-file workflow example

### Day 2: Advanced Patterns & Orchestration
**File:** [SESSION-2-advanced-patterns.md](SESSION-2-advanced-patterns.md)

**Duration:** 90 minutes
**Focus:** ReAct, Tree of Thoughts, Meta-prompting

**Demo repo branch:** [demo-day-2](https://github.com/josephrobertlopez/spring-migration-demo/tree/demo-day-2)
