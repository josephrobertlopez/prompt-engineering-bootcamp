# 5-Mode Prompt Templates

These templates structure AI interactions systematically for any use case.

## Usage

1. Copy all 5 files to your project
2. Fill in the templates with your specific context
3. Load files into your AI tool in order (0→1→2→3→4)
4. AI will generate solutions with full context

## Files

- `file-0-constitution.md` - Project rules (write once, use forever)
- `file-1-specification.md` - What to build (per component)
- `file-2-planning.md` - How to build it (per approach)
- `file-3-abcd.md` - Decisions (optional, as needed)
- `file-4-implementation.md` - Generate solution (every time)

## Reusability

- Mode 0: 100% reusable across project
- Mode 1: Template reusable, customize per component
- Mode 2: Reusable for similar components
- Mode 3: Use when decisions exist
- Mode 4: Template constant, just reference others
