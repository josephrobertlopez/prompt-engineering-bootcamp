This workshop teaches prompt engineering through a maturity lens:

- **游릭 Tier 1 (10+ years):** ADRs, RFCs, Few-shot, Chain-of-Thought, Persona, Template patterns
- **游리 Tier 2 (1-3 years):** .cursorrules, .github/copilot-instructions.md
